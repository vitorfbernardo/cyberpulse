Comandos Úteis

Atualizar Notícias

python manage.py update_news

Busca e importa novas notícias de todas as fontes RSS configuradas

Criar Migrações

python manage.py makemigrations

Cria arquivos de migração para alterações no banco de dados

Aplicar Migrações

python manage.py migrate

Aplica as migrações pendentes no banco de dados

Criar Superusuário

python manage.py createsuperuser

Cria um usuário administrador para acessar o painel admin

Iniciar Servidor

python manage.py runserver

Inicia o servidor de desenvolvimento Django

Limpar Cache

python manage.py clearcache

Limpa o cache do sistema (se configurado)

Configurações de Atualização Automática

Atualização Automática via APScheduler

Sistema configurado para buscar notícias automaticamente a cada 1 hora

ATIVO

Intervalo de Atualização

Frequência: A cada 1 hora (modificável em scheduler.py)

60 minutos

Compatibilidade

Funciona em Windows, Linux e Mac OS

Multiplataforma

Como funciona: O agendador APScheduler executa automaticamente python manage.py update_news a cada 1 hora quando o servidor Django está rodando.

Para alterar a frequência: Edite o arquivo news/scheduler.py e ajuste o parâmetro hours=1.

Iniciando automaticamente: O agendador inicia automaticamente com o comando python manage.py runserver.
